# **App Name**: Engineer Architects

## Core Features:

- Hero Section Display: Showcase a prominent hero section with the provided image, headline, subheadline, authority line, and calls to action.
- Engineering Mindset Section: Display the user's engineering philosophy and core focus areas in a clear, text-based section.
- Experience Timeline: Present professional experience using a clean, vertical timeline format.
- Portfolio Presentation Embed: Embed a placeholder PowerPoint file using Office Viewer in a responsive container, with a download button.
- Resume Embed: Embed a placeholder PDF resume via an iframe, with a strong 'Download Resume' call to action.
- Hobbies Section: Feature a hobbies section with a dummy image and related text labels.
- Static Contact Form: Implement a simple, frontend-only contact form with direct contact information display.
- SEO & Metadata Configuration: Implement all specified SEO requirements including title, meta description, and Open Graph tags for optimal search engine visibility.

## Style Guidelines:

- Primary content color: Deep blue-black (#0F172A) for strong, executive headlines and text on lighter backgrounds.
- Background color: A crisp, off-white (#F8FAFC) is used for the general site background, emphasizing cleanliness and readability.
- Accent color: A modern teal-green (#14B8A6) is provided to highlight call-to-action buttons and interactive elements, adding a touch of digital sophistication.
- Hero section gradient: A subtle transition from #0F172A to #111827 forms the dark, commanding background of the hero area, setting a serious tone.
- Headings font: 'Poppins' (sans-serif) for a precise, contemporary, and prominent display of titles. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) chosen for its excellent legibility and neutral, objective aesthetic, suitable for extensive body text. Note: currently only Google Fonts are supported.
- Use minimalistic and professional icons where necessary, aligning with the clean and executive aesthetic, but not required beyond simple functional symbols.
- Maintain strong whitespace and an executive clean layout, ensuring content breatheability and FAANG-level minimalism across all sections.
- Incorporate subtle fade-in animations on section scrolls or element appearances to enhance user engagement without distracting from the professional content.
